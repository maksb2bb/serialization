# Сериализация двусвязного списка с произвольными ссылками (ListRand)

Задание: реализация сериализации и десериализации двусвязного списка, где каждый узел содержит произвольную ссылку `rand`.

## Задача

Реализована сериализация и десериализация структуры `ListNode`, содержащей:
- Двусвязный список (`prev`, `next`)
- Произвольные ссылки `rand` на любой другой узел списка (или `nullptr`)
- Пользовательские данные `data` (строка)

## Форматы файлов

- **`inlet.in`** — текстовый входной файл
- **`outlet.out`** — бинарный выходной файл (сериализованный список)

## Структура проекта
```
├── TechTask.cpp
├── ListSerializer.h
├── ListSerializer.cpp
└── ListNode.h
```

## Основные возможности

- Чтение списка из `inlet.in`
- Построение двусвязного списка с корректными `prev`, `next` и `rand`
- Сериализация в компактный бинарный формат
- Десериализация из бинарного файла
- Корректная работа с `rand` указателями
- Освобождение памяти (`FreeList`)

## Формат бинарного файла (`outlet.out`)

1. `uint32_t` — количество узлов `N`
2. Для каждого узла:
   - `uint32_t` — длина строки `data`
   - `data` — байты строки (UTF-8)
   - `int32_t` — индекс узла, на который указывает `rand` (`-1` = nullptr)

## Как запустить

1. Поместите файл `inlet.in` в директорию с исполняемым файлом
2. Запустите программу
3. В результате будет создан файл `outlet.out`

## Пример `inlet.in`

```txt
apple;2
banana;-1
carrot;1
```
